Für mehr Sicherheit im Internet

So funktioniert der IT-Sicherheitscheck:

Wählen Sie zunächst das Gerät aus, das Sie für die Internetnutzung verwenden und für das Sie Handlungsempfehlungen erhalten möchten.

Hier können Sie nun aus verschiedenen typischen Online-Aktivitäten eine auswählen, zu der Sie Handlungsempfehlungen wünschen.

Hier haben Sie die Möglichkeit, anzugeben, welche IT-Sicherheitsmaßnahmen Sie bereits umgesetzt haben. Markieren Sie dazu die entsprechenden Maßnahmen. Falls Sie bislang noch keine Sicherheitsmaßnahmen ergriffen haben, wählen Sie einfach keiner der Option aus und lassen Sie die Checkboxen leer.
